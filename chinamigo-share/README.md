# ChinaMigo Native Site

This package contains the ChinaMigo website built with native HTML, CSS and JavaScript.

## Run locally

```bash
node server.js
```

Then open:

```text
http://127.0.0.1:4174
http://127.0.0.1:4174/trips
http://127.0.0.1:4174/trips/shanghai-lifestyle-trip
```

## Notes

- No React, Vue, Next, Vite or other frontend framework is required.
- Public content data is included in `data/`.
- Private inquiry records and email notification logs were intentionally excluded from this share package.
